# ⚖️ LexAI
## Asisten Riset Hukum Berbasis Artificial Intelligence


## Deskripsi

LexAI adalah website berbasis Artificial Intelligence yang dirancang untuk membantu mahasiswa Ilmu Hukum dalam melakukan riset, memahami materi hukum, menganalisis kasus sederhana, serta membuat referensi akademik.


## Tujuan Project

Project ini bertujuan memanfaatkan teknologi AI dalam bidang pendidikan hukum agar proses belajar dan pencarian informasi hukum menjadi lebih efektif, interaktif, dan mudah digunakan.


## Fitur

- 🤖 AI Legal Assistant
- 📚 Ringkasan Materi Hukum
- 📖 Kamus Istilah Hukum
- ⚖️ Analisis Kasus Sederhana
- 📄 Generator Referensi Akademik
- 🌙 Dark Mode
- 📱 Responsive Design
- ⬆️ Back to Top Button


## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript
- Artificial Intelligence API
- GitHub
